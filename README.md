# 🏨 King Sukh Guest House – Website Redesign

A modern and fully responsive redesign of the King Sukh Guest House website built using React, TypeScript, and Tailwind CSS.

## 🚀 Live Demo
🔗 https://your-vercel-link.vercel.app

## 🛠 Tech Stack
- React (Vite)

## ✨ Features
- Clean and modern UI
- Fully responsive design
- Room showcase section
- Image gallery
- Integrated contact form
- Google Maps location link
